enum FaultModel::FaultModelType : int
strong

Enumerates different fault models for ATPG (Automatic Test Pattern Generation).

The FaultModelType enum class defines different fault models that can be used in ATPG algorithms. Each fault model represents a specific type of fault that may occur in a digital circuit. The integer values associated with each fault model can be used for numerical comparisons or indexing.
